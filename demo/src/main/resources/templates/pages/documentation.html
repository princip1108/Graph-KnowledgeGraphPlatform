平台介绍

GraphWisdom智汇星图是一个知识图谱构建与可视化平台，帮助您创建、管理和探索知识网络。 平台支持 JSON 格式的图谱数据导入，提供力导向图可视化、节点关系编辑和社区交流等功能。

提示

建议新用户先完成账户设置，然后按照教程创建第一个示例图谱来熟悉平台功能。

主要功能特性

  • 支持 JSON 格式图谱数据导入，可导出为 JSON、CSV、PNG
  • 力导向图可视化，支持节点搜索、类型筛选
  • 在线编辑图谱：添加/修改/删除节点和关系
  • 图谱收藏、分享链接、状态管理（草稿/发布/私有）
  • 社区论坛：发帖交流、评论互动、私信功能

创建新图谱

在探索页点击"上传图谱"按钮，上传 JSON 格式的图谱文件，填写图谱名称、描述、选择领域分类，即可创建知识图谱。

常见问题解答

如何注册和创建账户？

注册账户非常简单，只需要提供邮箱地址和设置密码即可。

  1. 点击页面右上角的"注册"按钮
  2. 填写用户名和有效的邮箱地址
  3. 设置安全密码
  4. 勾选同意用户协议和隐私政策，完成注册
支持哪些数据格式导入？

平台目前支持 JSON 格式的图谱文件导入，文件需包含以下结构：

  • nodes：节点数组，每个节点包含 name（必填）、type、description
  • relations：关系数组，每条关系包含 source、target（节点名称匹配）、type
  • 可选字段：name（图谱名称）、description（图谱描述）、coverImage（封面URL）
如何分享我的知识图谱？

您可以通过以下方式分享您的知识图谱：

  • 将图谱状态设为"已发布"，其他用户即可在探索页看到
  • 通过分享链接将图谱发送给指定人员
  • 导出为 JSON、CSV 或 PNG 格式离线分享
